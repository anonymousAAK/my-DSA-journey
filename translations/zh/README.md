# My DSA Journey —— 从入门到研究进阶

> **30 周。4 种语言。真实的正确性测试、mastery quizzes，以及 4 小时的 quickstart —— 把 DSA 当作问题求解来学，而不是记忆。**

一个全面的、30 周结构化的数据结构与算法学习指南 —— 从第一性原理到研究级别的主题，再到面试 mastery。

所有实现都在**四种语言**中可用：Java、Python、C++、Rust —— 含详细注释、复杂度分析和测试用例。

---

## 从这里开始

**第一次来这个 repo？** 30 周是一项实打实的承诺。在做出承诺之前，先试试下面两个 on-ramp 之一：

- **[QUICKSTART.md](QUICKSTART.md) —— 4 小时，8 道题。** 一个聚焦 session，带你在每个难度级别的代表性题目上走一遍完整方法论（分解 → 识别模式 → 实现 → journal）。如果这个循环对你有效，长路径不过是它的更多次重复。
- **[diagnostic.md](diagnostic.md) —— 15 道题，~25 分钟。** 一个校准过的 diagnostic，把你放到课程里的合适位置，避免花几周重学已会的东西。把你的分数映射到具体的起始周。

已经决定了？直接跳到 [Learning Paths](#learning-paths) 或直接打开 [Week 1](../../Week%201/)。

**找特定文档？** 看 [INDEX.md](../../INDEX.md) —— repo 中每个文件的主索引，按用途分组。

## 如何使用本指南

1. **新手**：从 Week 1 开始，按顺序推进。不要跳周。
2. **中级**：按舒适度，跳到 Week 8（Search）或 Week 11（链表）。
3. **进阶**：Weeks 18-24 覆盖 DP、高级图论和研究级主题。
4. **面试备战**：先走上面的 Interview Prep 路径，然后做 Week 30。
5. 每个 Week 文件夹组织成 `java`、`python`、`cpp`、`rust`、`web` 子文件夹，每种语言每个主题有一个编号文件。
6. **推荐的学习顺序**：读 Java 文件以获得 canonical walkthrough → 读同号的 Python/C++/Rust/HTML 文件 → 四个都跑一下对比输出 → 看文件末的 `NOTES` block 了解 idiomatic 差异。
7. 每个文件包含：问题描述 → 思路/伪代码 → 实现 → 复杂度分析。
8. **学语言的人**：跨语言对比实现，理解设计差异。
9. **打开 HTML 页**：每周有 `web/index.html`，含交互式讲解、图示，以及可复制的代码。

10. 每个 `Week N/README.md` 现在镜像 Java 主题路线图，并为 Python/C++/Rust 定义了"detailed parity"的含义。
