# Mentor Directory

This is the public list of people who have opted into mentoring learners
working through the curriculum. See [`docs/MENTORSHIP.md`](../docs/MENTORSHIP.md)
for what mentorship looks like, eligibility, and norms.

## How to get listed

1. Open a [Mentor Signup issue](../../issues/new?template=mentor_signup.yml).
2. A maintainer reviews eligibility (Foundation certification or above) and
   merges a PR adding you to the table below.

## How to get unlisted

Open a PR removing your row, or comment on your original signup issue and a
maintainer will open the PR for you.

## Directory

The directory is empty at the program's launch — be the first to sign up.

### Column reference

- **Handle**: GitHub handle or display name. Links to a profile if available.
- **Topics**: which weeks / specific topics the mentor is comfortable on.
- **Cert level**: Foundation / Practitioner / Specialist / Expert (see
  `docs/MASTERY.md`).
- **Timezone**: e.g. `UTC+5:30` or `America/Los_Angeles`.
- **Contact**: Discord handle, email, or other direct channel.
- **Office hours?**: link to the mentor's recurring GitHub Discussions
  office-hours thread if they run one; `—` otherwise.
