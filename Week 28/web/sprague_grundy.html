Sprague-Grundy Theorem

WEEK 28 - WEB (JS) ADVANCED TOPICS

CONCEPT: Every impartial game has a Grundy number g(s) = mex of children's Grundy numbers. Multi-pile sums XOR their Grundy values; losing iff XOR=0.

KEY POINTS: Generalises Nim's XOR rule; DP over single-pile states; XOR for sums.

ALGORITHM: Build g[0..n] via mex of reachable predecessors; XOR g[piles[i]] for multi-pile decision.

JS-SPECIFIC NOTES: Set for the reachable Grundy values; iterate from 0 to find mex.

DRY RUN: moves {1,3,4}: g(10)=1; two piles [10,6]: XOR(1,2)=3 != 0 -> first wins.

COMPLEXITY: Time O(n * |moves|); Space O(n).
