Minimax (Tic-Tac-Toe)

WEEK 28 - WEB (JS) ADVANCED TOPICS

CONCEPT: Two-player zero-sum game search. Max picks the max over child values; Min picks the min. Returns the value with optimal play.

KEY POINTS: O(b^d); memoise on hashable state; for Tic-Tac-Toe d <= 9 so brute force is fine.

ALGORITHM: recurse to terminals; backpropagate max/min.

JS-SPECIFIC NOTES: Use a string board for fast Map keys; spread/slice to clone the board.

DRY RUN: Empty board (X to move) -> value 0 (draw with optimal play).

COMPLEXITY: Time O(b^d); Space O(d) + memo.
