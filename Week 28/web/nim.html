Nim Game

WEEK 28 - WEB (JS) ADVANCED TOPICS

CONCEPT: Two players alternate; on each turn remove any positive number of stones from a single pile. Loser is the player unable to move. Bouton's theorem: first player wins iff XOR of piles is non-zero.

KEY POINTS: XOR is the invariant; from XOR != 0 the mover can always move to XOR == 0.

ALGORITHM: xor_sum = XOR(piles); decide winner; constructively find a pile p with (p XOR xor_sum) < p.

JS-SPECIFIC NOTES: JS bitwise operators coerce to 32-bit ints; safe for typical pile sizes.

DRY RUN: piles [3,4,5] -> XOR = 2 -> first wins; reduce pile 0 from 3 to 1 (3^2=1).

COMPLEXITY: Time O(n); Space O(1).
