Alpha-Beta Pruning

WEEK 28 - WEB (JS) ADVANCED TOPICS

CONCEPT: Minimax with branch pruning. Maintain a window [alpha, beta]; abandon a subtree when the running v cannot affect the parent's choice.

KEY POINTS: alpha = best for Max so far; beta = best for Min so far; prune when alpha >= beta.

ALGORITHM: see code: identical to minimax but tracks alpha/beta and breaks on cutoff.

JS-SPECIFIC NOTES: Use Number.NEGATIVE_INFINITY / Number.POSITIVE_INFINITY for initial bounds.

DRY RUN: Empty TTT board -> value 0 with ~5-7x fewer node visits than plain minimax. The output prints the node counter.

COMPLEXITY: Time O(b^d) worst; O(b^(d/2)) with good ordering. Space O(d).
