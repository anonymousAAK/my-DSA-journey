Merge Intervals Pattern

WEEK 30 - WEB (JS) ADVANCED TOPICS

CONCEPT: Sort by start, single linear pass to merge overlapping intervals, insert a new interval, count meeting rooms, etc.

KEY POINTS: Sort then merge; insert in 3 passes; meeting rooms via sorted starts/ends.

ALGORITHM: mergeIntervals, insertInterval, minMeetingRooms, canAttendAll.

JS-SPECIFIC NOTES: Tuples are arrays; sort with (a,b)=>a[0]-b[0] for start order.

DRY RUN: merge [[1,3],[2,6],[8,10],[15,18]] -> [[1,6],[8,10],[15,18]]; insert [2,5] into [[1,3],[6,9]] -> [[1,5],[6,9]].

COMPLEXITY: Merge / Rooms O(n log n); Insert O(n).
