# Video Walkthroughs

> **Status**: Scaffolding incomplete (translation/audio agent hit a session limit before producing scripts). PRs welcome.

The plan is six 5-minute video scripts (one per curriculum phase) ready for
someone to record via OBS / Loom / Screenflow.

## To contribute

1. Pick a phase (1-6).
2. Write a 5-minute screenplay-style script in `video/scripts/phase_N.md`
   with VISUAL CUES annotated for screen recording.
3. Record (any tool). Upload to YouTube / Vimeo.
4. PR the script + a link to the recording.
