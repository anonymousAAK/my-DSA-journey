Week 4

Pattern Programming

Master nested loops by printing star, number, and alphabet patterns -- a fundamental skill for understanding loop control and building complex logic.

Beginner ~6 hours estimated 8 patterns covered

🎯 Learning Objectives

  • Understand nested loop structure: outer loop for rows, inner loop(s) for columns
  • Identify the relationship between row number and what gets printed
  • Print square, triangular, and pyramid patterns with stars and numbers
  • Use ASCII arithmetic to generate alphabet-based patterns
  • Handle leading spaces for right-aligned and mirror-image patterns
  • Recognize that ALL patterns follow the same outer-loop/inner-loop template

💡 The Pattern-Solving Approach

Every pattern problem follows the same template. The key is identifying three things for each row:

Step 1: How many rows? → Outer loop: for i = 1 to N Step 2: For each row, what is printed? - How many leading spaces? → First inner loop - What characters/numbers? → Second inner loop - How many of them? → Inner loop bound Step 3: What depends on the row number (i)? - Number of items per row - The character or number printed - The number of spaces before content
Key Insight
In most patterns, the inner loop bound depends on the outer loop variable i. This is what creates the triangular or pyramid shape. If the inner loop runs exactly N times regardless of i, you get a rectangle.

⬜ Square Pattern

Print an N x N grid where every cell contains the value N. Both loops run exactly N times.

Output (N=4)

4444 4444 4444 4444

Implementation

Square_Pattern.java
import java.util.Scanner; public class Solution { public static void main(String[] args) { Scanner s = new Scanner(System.in); int n = s.nextInt(); for (int i = 1; i <= n; i++) { // rows for (int j = 1; j <= n; j++) { // columns (always N) System.out.print(n); // print N at every position } System.out.println(); // newline after each row } } } // Time: O(N^2), Space: O(1)
patterns.cpp
// Square pattern: N x N grid int n = 4; for (int i = 0; i < n; i++) { for (int j = 0; j < n; j++) cout << n; cout << endl; }
patterns.py
def square_pattern(n: int): for _ in range(n): print(" ".join([str(n)] * n)) square_pattern(4) # 4 4 4 4 # 4 4 4 4 # 4 4 4 4 # 4 4 4 4
patterns.rs
let n = 4; for _ in 0..n { for _ in 0..n { print!("{}", n); } println!(); }

★ Triangle Star Pattern

Print a right-angled triangle of stars. Row i contains exactly i stars -- the inner loop bound is i, not N.

Output (N=4)

* ** *** ****

Implementation

Tri_Star_Pattern.java
import java.util.Scanner; public class Solution { public static void main(String[] args) { Scanner s = new Scanner(System.in); int n = s.nextInt(); for (int i = 1; i <= n; i++) { // rows: 1 to N for (int j = 1; j <= i; j++) { // columns: 1 to i (not N!) System.out.print('*'); } System.out.println(); } } }
patterns.cpp
void rightTriangleStars(int n) { for (int i = 1; i <= n; i++) { for (int j = 0; j < i; j++) cout << "* "; cout << endl; } }
patterns.py
def right_triangle_stars(n: int): for i in range(1, n + 1): print("* " * i) # One-liner: print("\n".join("*" * i for i in range(1, 6)))
patterns.rs
fn right_triangle_stars(n: usize) { for i in 1..=n { for _ in 0..i { print!("* "); } println!(); } } // Compact: println!("{}", "* ".repeat(i));

🅰 Alphabet Pattern

Row i prints the i-th letter of the alphabet repeated i times. Uses ASCII arithmetic: ch = 'A' + (row - 1).

Output (N=4)

A BB CCC DDDD

Implementation

Alpha_Pattern.java
int currRow = 1; while (currRow <= n) { int currCol = 1; char ch = (char) ('A' + currRow - 1); // row 1->'A', 2->'B', etc. while (currCol <= currRow) { System.out.print(ch); // same char across row currCol += 1; } System.out.println(); currRow += 1; }
patterns.cpp
void alphabetTriangle(int n) { for (int i = 0; i < n; i++) { char ch = 'A' + i; for (int j = 0; j <= i; j++) cout << ch << " "; cout << endl; } }
patterns.py
def alphabet_triangle(n: int): for i in range(n): ch = chr(ord('A') + i) # ASCII arithmetic print(f"{ch} " * (i + 1)) alphabet_triangle(4)
patterns.rs
fn alphabet_triangle(n: usize) { for i in 0..n { let ch = (b'A' + i as u8) as char; for _ in 0..=i { print!("{} ", ch); } println!(); } }

🔡 Character Pattern (Consecutive)

Unlike the Alphabet Pattern where each row repeats one letter, here each row prints consecutive letters. Row i starts at 'A' + (i-1) and prints i consecutive characters.

Output (N=4)

A BC CDE DEFG

Implementation

Char_Pattern.java
int currRow = 1; while (currRow <= n) { int currCol = 1; char ch = (char) ('A' + currRow - 1); // starting char for this row while (currCol <= currRow) { System.out.print((char)(ch + currCol - 1)); // consecutive chars currCol += 1; } System.out.println(); currRow += 1; } // Row 3 starts at 'C', prints: C D E
patterns.cpp
for (int i = 0; i < n; i++) { char start = 'A' + i; for (int j = 0; j <= i; j++) cout << (char)(start + j); cout << endl; }
patterns.py
for i in range(n): start = ord('A') + i print("".join(chr(start + j) for j in range(i + 1)))
patterns.rs
for i in 0..n { let start = b'A' + i as u8; for j in 0..=i { print!("{}", (start + j as u8) as char); } println!(); }
Key Insight
The difference between Alpha Pattern and Char Pattern is subtle but important: in the Alpha Pattern, the character depends only on the row ('A' + row - 1). In the Char Pattern, it depends on both row and column ('A' + row - 1 + col - 1).

📜 Mirror Image Pattern

A right-aligned triangle requires two inner loops per row: one for leading spaces and one for the numbers. The space count decreases as the row number increases.

Output (N=4)

1 12 123 1234 Spaces per row: (N - row) spaces, then numbers 1 to row Row 1: 3 spaces + "1" Row 2: 2 spaces + "12" Row 3: 1 space + "123" Row 4: 0 spaces + "1234"

Implementation

Mirror_Image.java
int currRow = 1; while (currRow <= n) { // First inner loop: print leading spaces int spaces = 1; while (spaces <= n - currRow) { System.out.print(" "); spaces += 1; } // Second inner loop: print numbers 1 to currRow int currCol = 1; while (currCol <= currRow) { System.out.print(currCol); currCol += 1; } System.out.println(); currRow += 1; }
patterns.cpp
void numberPyramid(int n) { for (int i = 1; i <= n; i++) { cout << string(n - i, ' '); // leading spaces for (int j = 1; j <= i; j++) cout << j << " "; cout << endl; } }
patterns.py
def right_aligned_numbers(n: int): for i in range(1, n + 1): spaces = " " * (n - i) nums = " ".join(str(j) for j in range(1, i + 1)) print(spaces + nums) right_aligned_numbers(4)
patterns.rs
fn number_pyramid(n: usize) { for i in 1..=n { print!("{}", " ".repeat(n - i)); // leading spaces for j in 1..=i { print!("{} ", j); } println!(); } }

💎 Advanced Patterns

Once you master the basic patterns, try these more challenging variations that combine multiple techniques.

Diamond Pattern (N=4)

* * * * * * * * * * * * * * * *

Butterfly Pattern (N=4)

* * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * * *
Advanced Patterns
// Diamond: upper half + lower half static void diamond(int n) { // Upper half (pyramid) for (int i = 1; i <= n; i++) { for (int s = 0; s < n - i; s++) System.out.print(" "); for (int j = 0; j < i; j++) System.out.print("* "); System.out.println(); } // Lower half (inverted pyramid) for (int i = n - 1; i >= 1; i--) { for (int s = 0; s < n - i; s++) System.out.print(" "); for (int j = 0; j < i; j++) System.out.print("* "); System.out.println(); } }
patterns.cpp
void butterfly(int n) { // Upper half for (int i = 1; i <= n; i++) { for (int j = 0; j < i; j++) cout << "* "; for (int j = 0; j < 2*(n-i); j++) cout << " "; for (int j = 0; j < i; j++) cout << "* "; cout << endl; } // Lower half (mirror) for (int i = n; i >= 1; i--) { for (int j = 0; j < i; j++) cout << "* "; for (int j = 0; j < 2*(n-i); j++) cout << " "; for (int j = 0; j < i; j++) cout << "* "; cout << endl; } }
patterns.py
def diamond(n: int): for i in range(1, n + 1): print(" " * (n - i) + "* " * i) for i in range(n - 1, 0, -1): print(" " * (n - i) + "* " * i) def butterfly(n: int): for i in range(1, n + 1): print("* " * i + " " * (n - i) + "* " * i) for i in range(n, 0, -1): print("* " * i + " " * (n - i) + "* " * i)
patterns.rs
fn diamond(n: usize) { for i in 1..=n { print!("{}", " ".repeat(n - i)); for _ in 0..i { print!("* "); } println!(); } for i in (1..n).rev() { print!("{}", " ".repeat(n - i)); for _ in 0..i { print!("* "); } println!(); } }
Pattern Strategy
For complex patterns like diamond and butterfly, split them into upper and lower halves. Each half is a simpler pattern you already know how to print. Compose simple patterns to build complex ones.

🚀 Week 4 Summary

  • All patterns use the same template: outer loop for rows, inner loop(s) for columns
  • Square: inner loop bound = N (constant)
  • Triangle: inner loop bound = i (row number)
  • Right-aligned: requires two inner loops -- one for spaces, one for content
  • Alphabet patterns: use ASCII arithmetic: 'A' + offset
  • Complex patterns: decompose into simpler halves (upper/lower)
  • Key question: for each row, what depends on the row number i?
