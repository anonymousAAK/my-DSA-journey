Edmonds-Karp Maximum Flow

WEEK 26 - WEB (JS) ADVANCED TOPICS

CONCEPT: Ford-Fulkerson with BFS path discovery; the shortest augmenting path is selected each round, giving O(V * E^2) total work independent of edge capacities.

KEY POINTS: BFS shortest path; parent[] / parentEdge[] for reconstruction; updates both forward and reverse residuals.

ALGORITHM: while BFS finds path s -> t with positive residual, compute bottleneck and update both directions. Stop when no augmenting path exists.

JS-SPECIFIC NOTES: JS arrays make great BFS queues when used with shift()/push(); for big graphs, prefer index-based ring buffers since shift() is O(n).

DRY RUN: CLRS graph -> 0-1-3-5 (12), 0-2-4-5 (4), 0-2-4-3-5 (7) -> total 23.

COMPLEXITY: Time O(V * E^2); Space O(V + E).
