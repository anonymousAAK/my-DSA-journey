Sweep Line - Skyline & Max Overlap

WEEK 27 - WEB (JS) ADVANCED TOPICS

CONCEPT: A conceptual vertical sweep line crosses the plane, processing events sorted by x while maintaining an active set.

KEY POINTS: For skyline use a max-heap of active heights; for interval overlap use a running +1/-1 counter.

ALGORITHM: skyline: emit start/end events; track max height via heap; output (x, h) when h changes. overlap: sort signed events; running sum tracks simultaneous active count.

JS-SPECIFIC NOTES: JS lacks a heap; we hand-roll a small max-heap class. Sorting tuples works via comparator function.

DRY RUN: buildings = [(2,9,10),(3,7,15),(5,12,12), (15,20,10),(19,24,8)] -> skyline [(2,10),(3,15),(7,12),(12,0),(15,10), (20,8),(24,0)].

COMPLEXITY: Time O(n log n); Space O(n).
