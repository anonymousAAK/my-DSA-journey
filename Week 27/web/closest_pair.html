Closest Pair of Points (Divide & Conquer)

WEEK 27 - WEB (JS) ADVANCED TOPICS

CONCEPT: Divide-and-conquer to find the smallest pairwise Euclidean distance among n points in O(n log n).

KEY POINTS: sort by x; recurse; combine via a strip of width 2d sorted by y; each strip point looks at ~7 neighbours.

ALGORITHM: base case <=3 -> brute; else recurse on halves and merge via strip scan.

JS-SPECIFIC NOTES: Math.hypot is numerically stable. Use slice() to copy point ranges since we mutate locally.

DRY RUN: [(2,3),(12,30),(40,50),(5,1),(12,10),(3,4)] -> ~1.4142 (between (2,3) and (3,4)).

COMPLEXITY: Time O(n log n); Space O(n).
